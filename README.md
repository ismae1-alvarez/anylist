# Dev 

1. Clonar el proyecto
2. Copiar el 
```
env.template
``` 
y renombrar a 
```
env.
```
3. Ejecutar 
```
pnpm install
```
4. Levantar la imagen (Docker desktop) 
```
docker-compose up -d
```
5. Levantar el backen de Nest 
```
pnpm dev:start
```
6. Visita el sitio 
```
localhost:3000/graphql
```
7. Ejecuta la __"mutation"__ executeSeed, para llenar la base de datos con informacion
